# EcoParking Web (Frontend)

Este repositorio contiene la interfaz web administrativa para el sistema EcoParking. Está construida con React + vite.

---

## Instalación y Ejecución

Sigue estos pasos uno por uno para ejecutar el proyecto en tu entorno local.

### 1. Clonar el repositorio
Copia el proyecto a tu máquina local:
```bash
git clone https://github.com/Jonathancraxker/ecoparking_web.git
```
### 2. Entrar a la carpeta del proyecto
```bash
cd nombre_proyecto
```
### 3. Instalar dependencias
```bash
npm install
```
### 4. Inicializar aplicación (entorno local)
```bash
npm run dev
```
### Principales librerías utilizadas:
- react-router-dom: Manejo de rutas y navegación entre páginas.
- axios: Cliente HTTP para realizar peticiones al Backend (API).
- bootstrap / react-bootstrap: Framework de estilos y componentes UI responsivos.
- sweetalert2: Para mostrar alertas y modales interactivos y estéticos.
- qrcode.react: Generación de códigos QR para los accesos.
- vite: Herramienta de compilación y servidor de desarrollo rápido.
